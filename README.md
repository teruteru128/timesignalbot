# Time signal bot

Time signal bot for Discord

![ロゴ](https://cdn.discordapp.com/attachments/879315010218774531/914442683697745940/WS000353.png "時報bot")

ごく普通の時報botです。
真夜中に時報をお届けします。

ごくたまに他の時報を話すこともあります。

## その他コマンド

プレフィックスは `/` です。

- neko      見て！ねこがいるよ　かわいいね　隠しあり
- nyanpass  にゃんぱすーボタンのカウンターを取得して表示するのん
- help      コマンドのヘルプを表示します。
